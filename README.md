#### 启动项目
    - git clone 
    - npm install
    - mongod --dbpath /xxx(项目文件目录)/react-express-mongodb/app/server/db/ 或者 直接启动mongod（自行Google）
    - 前端启动 npm start 
    - 后端启动 npm run server
